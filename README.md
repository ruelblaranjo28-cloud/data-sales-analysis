# 🚗 Automobile Sales Data Analysis & Visualization

🚀 This project demonstrates data analysis and visualization skills using real-world automobile sales data to uncover trends and business insights.

## 📊 Exploratory Data Analysis (EDA)
### By Ruel Laranjo

---

## 📌 Introduction

This project focuses on analyzing automobile sales data to uncover meaningful patterns and trends over time.  
The analysis emphasizes understanding how sales behave under different economic conditions, including recession periods.

---

## 🎯 Objectives

The main goals of this project are:

- Clean and prepare the dataset for analysis  
- Perform Exploratory Data Analysis (EDA)  
- Create meaningful visualizations  
- Identify trends and patterns in automobile sales  

---

## 🛠️ Tools & Technologies

- Python 🐍  
- Pandas  
- Matplotlib  
- Seaborn  
- Plotly  
- Folium (for map visualization)  

---

## 📂 Dataset

The dataset is loaded from an online source and contains information such as:

- Date  
- Year and Month  
- Automobile Sales  
- Vehicle Type  
- Economic Indicators (GDP, Consumer Confidence, etc.)

- 📉 Average Automobile Sales by Year

---

## 🔍 Key Insights

- Automobile sales show noticeable fluctuations over time  
- Sales tend to decline during recession periods  
- Different vehicle types perform differently depending on economic conditions  

---

## 🧾 Results

The analysis highlights key trends in automobile sales and shows how external factors like economic conditions affect performance.

---

## ✅ Conclusion

From the analysis:

- Automobile sales fluctuate over time  
- Recession periods significantly impact sales  
- Certain vehicle types perform better than others  
- Geographic distribution shows variation in sales  

This project demonstrates how data visualization helps in understanding business trends.
